## 🚀 快速部署

### 方式 1：Cloudflare Pages（推荐）
```
Cloudflare → Workers & Pages → Create → Pages → Upload assets
项目名：tiktok-money-calculator
上传整个文件夹 → Deploy → 绑定自定义域名
```

### 方式 2：Vercel
```bash
cd tiktok-money-calculator && npx vercel --prod
```

### 方式 3：Netlify
```bash
npx netlify deploy --prod --dir=.
```
